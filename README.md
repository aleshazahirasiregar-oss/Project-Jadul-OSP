# Project Jadul OSP

**"support ver Android 5.0 (Lolipop) hingga Android 15, One Device at a Time."**

Selamat datang di repository resmi Project Jadul OSP. Proyek ini didedikasikan untuk menyelamatkan perangkat legacy dan low-end agar tetap relevan di dunia digital. Kami mengambil perangkat lunak lawas yang sudah tidak didukung dan mengubahnya menjadi Open Source Project (OSP) yang ringan, efisien, dan fungsional.

### 🛡️ Visi Garuda Xyber
Di tengah gempuran aplikasi modern yang rakus sumber daya, banyak perangkat low-end di luar sana yang menjadi korban bloatware. Proyek ini hadir untuk membebaskan perangkat tersebut dengan memberikan alternatif yang ringan dan tidak membebani sistem.

### ⚙️ Spesifikasi Teknis (The "Lightweight" Standard)
Kami menerapkan protokol ketat untuk menjaga performa perangkat tetap optimal:
*   Cache Management: Terkontrol ketat untuk stabilitas maksimal.
    *   Min Cache: 0,79 KB
    *   Max Cache: 1,2 KB
*   Architecture: Automated Cloud Build via GitHub Actions. Tidak ada beban kompilasi lokal pada perangkat.

### 🚀 Keunggulan Proyek
1.  Ringan: Dibuat khusus untuk hardware dengan RAM terbatas.
2.  Open Source: Transparan, bisa diaudit, dan dikembangkan bersama komunitas.
3.  Efisiensi: Tidak bergantung pada layanan sistem yang berat.
